# Calib

An sklearned based Spectrometer Calibration Package for HRS


## Spectrometer Calibration Pipeline
Currently it can handle the following jobs

* feature engineering 
* feature selection
* model training 
* model profiling
* model visualization
* model deployment 

## before get start

```
pip install --update SpectroCalib
```
